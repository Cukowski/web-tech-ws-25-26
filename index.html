Web Tech Notes (Worksheet 1)

IPv4 vs IPv6

IPv4: 32-bit, dotted decimal, e.g. 203.0.113.42

IPv6: 128-bit, colon-separated, e.g. 2001:db8:85a3::8a2e:370:7334

Why IPv6

  • Address exhaustion in IPv4
  • Better end-to-end connectivity
  • Modern features (autoconfiguration, IPsec support)
